# Food Menu Images

Place your food item images here.

## Naming Convention

Images must match the item `id` from the menu:
- `b_eggs.jpg` → item with id `"b_eggs"`
- `l_combo.jpg` → item with id `"l_combo"`
- etc.

## Image Specs

- **Format**: JPG or PNG
- **Size**: 400x400px (square) recommended
- **File size**: < 200KB per image

## Adding Images

1. Save image with correct name (e.g., `b_eggs.jpg`)
2. Place in this folder
3. Image will automatically appear in the UI

See `MENU_IMAGES_GUIDE.md` for full details.
